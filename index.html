Flappy Bird Clone

My implementation of flappy bird in Javascript. The right side shows the local highest three scores and the scores of the last five runs underneath that.

A* Algorithm Visualized

An implementation of the A star path finding algorithm visualized in Javascript. You can set the start and end positions and click and hold to draw boundaries. I imported a heap library to help with the implementation.

Image Convolution

A vanilla javascript implementation of a 3x3 kernel image convolution. You can select common kernels to apply or use a user inputted kernel. The bottom shows a histogram of the color channels that compose the selected image.

Weather Web App

Used the OpenWeather API to create a JS web app to get the weather.

Kirby SystemVerilog

I wrote it in SystemVerilog for the DE-115 FPGA. Background images was compressed with Python scripts into 8 bit format and flashed into 2MB SRAM while character images were programmed into on-chip memory. Most things including the game logic was written in Verilog. The exception was the keyboard polling. The health system was not fully debugged so chracters can't "die".

Lane Detection

A project I did college for lane detection and visualization in Python. OpenCV was used for image filtering and polynomial fitting was used to visualize the road.
