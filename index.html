Flappy Bird Clone

My implementation of flappy bird in Javascript. The right side shows the local highest three scores and the last five runs underneath that.

A* Algorithm Visualized

An implementation of the A star path finding algorithm visualized in Javascript. I used an external heap library to help with the implementation.

Image Convolution

A javascript implementation of a 3x3 kernel image convolution. You can select common kernels to apply or use a user inputted kernel. The bottom shows a histogram of the color channels that compose the selected image.

Weather Web App

Used the OpenWeather API to create a JS web app to get the weather.

Kirby SystemVerilog

I wrote it in SystemVerilog for the DE-115 FPGA. Background were compressed with Python scripts and flashed into 2MB SRAM while character images were programmed into on-chip memory. The health system was not fully debugged so chracters can't "die".

Lane Detection

A cool project in school for lane detection and visualization in Python.
